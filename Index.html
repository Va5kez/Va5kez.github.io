Speed is Key: The Game

There is a timer in the screen that indicates you how much time
left you have for reaching the end of each the level.

Every time you have reached the end of the level, it will load the next one.
The goal here is to run as fast as possible in the shortest
time possible, if you don't reach it in time is "GAME OVER".
The fastes you do it, greater your highscore will be.

Good luck :3 !!!!!
